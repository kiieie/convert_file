# 🎨 ConvertFile — 현대적/기능 중심 UI 리디자인 및 명세서 (UI Spec)

> 레트로 감성의 Windows XP 테마를 걷어내고, 오직 성능과 기능에 초점을 맞춘 미니멀 플랫(Minimal Flat) 스타일의 현대적인 직관적 UI/UX 설계 문서.  
> 불필요한 장식을 배제하여 AI 템플릿 느낌을 완전히 없애고, 전문적인 웹 유틸리티 서비스(Developer Tools 감성)의 느낌을 구현합니다.

---

## 1. 디자인 시스템 핵심 컨셉 (Design Tokens)

### 1.1 컬러 팔레트 (Color Palette)
AI 생성 페이지의 전형적인 스타일(파스텔톤의 지나친 둥글기, 복잡한 그라디언트)을 방지하고, GitHub/Vercel 디자인 느낌의 고대비 모노톤 및 신뢰성 있는 포인트 블루를 씁니다.

| 역할 | 컬러 칩 | Hex Code | 용도 |
| :--- | :--- | :--- | :--- |
| **Primary (Brand)** | Deep Indigo | `#0F172A` | 주요 헤더, 묵직한 텍스트 및 기본 배경선 |
| **Accent** | Royal Blue | `#2563EB` | 상호작용 액션 버튼, 포커스 아웃라인, 주요 링크 |
| **Success** | Green | `#16A34A` | 처리 완료 상태 표시, 정상 구동 알림 |
| **Error** | Red | `#DC2626` | 에러 알림 및 입력 유효성 실패 표기 |
| **Background (Light)** | Ice Gray | `#F8FAFC` | 전체 백그라운드 |
| **Card / Surface** | Clean White | `#FFFFFF` | 워크스페이스 보드, 파일 업로드 카드 |
| **Border / Stroke** | Cool Slate | `#E2E8F0` | 구분선 및 인풋 테두리 (1px 고체선 고정) |

### 1.2 타이포그래피 (Typography)
- **메인 폰트**: `Inter`, `System-ui` (애플/윈도우 공통 웹세이프 샌스세리프 폰트 패밀리)
- **가독성 규칙**: 
  - 본문 크기: `14px` (라인 높이 `1.5`)
  - 폼 레이블: `12px` (굵게, 대문자/회색조)
  - 툴 이름 타이틀: `20px` (세미볼드)

## 2. 전체 레이아웃 아키텍처 (Layout Structure)

새로운 사이트는 사용자가 진입하자마자 모든 도구를 한 눈에 파악할 수 있는 **2열 사이드바 레이아웃 (Sidebar-Main Workspace)** 구조를 채택합니다.

```
┌──────────────────────────────────────────────────────────────────┐
│ 🌐 ConvertFile (헤더)                                             │
├──────────────────────┬───────────────────────────────────────────┤
│ 📁 이미지 변환        │ 🛠️ 이미지 크기 조절 (Workspace)            │
│ ├ JPG to PNG         │ ┌───────────────────────────────────────┐ │
│ └ WebP Converter     │ │                                       │ │
│                      │ │        [ 파일 업로드 드롭존 ]          │ │
│ 📁 이미지 편집        │ │                                       │ │
│ ├ 크기 조절 (Resize) │ └───────────────────────────────────────┘ │
│ ├ 자르기 (Crop)      │ ┌───────────────────┐ ┌───────────────────┐ │
│ └ 회전 (Rotate)      │ │ 설정 카드 1       │ │ 설정 카드 2       │ │
│                      │ └───────────────────┘ └───────────────────┘ │
│ 📁 유틸리티          │ ┌───────────────────────────────────────┐ │
│ ├ QR/Barcode         │ │ 💾 변환 실행 버튼                     │ │
│ └ EXIF 지우개        │ └───────────────────────────────────────┘ │
└──────────────────────┴───────────────────────────────────────────┘
```

### 2.1 사이드바 (Navigation Sidebar)
- 화면 좌측에 고정(Fixed)되며, 모든 카테고리와 툴 목록을 직관적인 파일 트리 또는 카드 목록 형태로 나열합니다.
- 현재 사용 중인 도구는 활성화 표시(`font-weight: 600`, 왼쪽 3px 블루 인디케이터 라인)가 들어갑니다.

### 2.2 메인 워크스페이스 (Workspace)
- 선택한 도구가 단일 페이지 카드 형식으로 정갈하게 로드됩니다.
- **상단부**: 툴 타이틀 및 간결한 한 줄 설명.
- **중앙부 (2열 레이아웃)**:
  - *좌측*: 직관적인 파일 드롭존 또는 이미지 미리보기/캔버스 제어 영역.
  - *우측*: 미니멀한 설정 제어 상자(카드 단위로 구별).
- **하단부**: 변환 실행 버튼(가로 전체 너비 확보로 직관적인 클릭 지원) 및 텍스트 안내/FAQ.

## 3. UI 컴포넌트 세부 디자인 규격

### 3.1 파일 업로드 드롭존 (`.modern-upload-zone`)
- **디자인**: 2픽셀 대시 형태의 슬레이트 블루 보더선(`#94A3B8`), 넓은 안쪽 여백(`padding: 60px 40px`), 둥근 모서리(`border-radius: 8px`).
- **인터랙션**: 
  - 기본 상태에서도 파란색 버튼과 깔끔한 파일 로드 영역이 눈에 띄어 파일 탐색이 쉬움.
  - 마우스 드래그 오버 시 백그라운드가 부드러운 스카이 블루(`#EFF6FF`)로 스위칭되며 보더선이 굵은 Royal Blue(`#2563EB`)로 자동 하이라이트.
  - 드롭존 영역 내부의 어떤 곳을 클릭해도 파일 찾기 팝업 창이 즉시 실행되도록 마우스 이벤트를 완벽하게 바인딩.
- **아이콘**: AI 템플릿의 형형색색 3D 일러스트 대신 미니멀하고 가독성 좋은 36px 스레이트색 파일 업로드 아이콘 배치.

### 3.2 입력 폼 및 컨트롤 (`.modern-input`, `.modern-select`)
- 입체 베벨 그림자를 완전히 제거하고 Flat한 1px 단색 테두리로 전환.
- 포커스 시 브라우저 기본 아웃라인을 끄고 `box-shadow: 0 0 0 2px rgba(37, 99, 235, 0.2)` 및 블루 보더 적용.

### 3.3 실행 액션 버튼 (`.modern-btn`)
- **기본 버튼**: Deep Indigo 백그라운드, 흰색 텍스트, 6px 라운딩. 호버 시 살짝 밝아짐.
- **세컨더리 버튼**: 테두리만 있는 아웃라인 버튼 (`border: 1px solid #E2E8F0`).
- **애니메이션**: 마우스 호버 시 `translateY(-1px)` 및 미세한 그림자 확대 효과 적용(0.15s 속도).

### 3.4 탭 및 탐색 메뉴 UI (`.modern-tabs`)
- 두 개 이상의 서브 메뉴가 필요한 툴(예: 바코드 생성/스캔, PDF 변환/추출)은 상단에 깔끔한 세그먼트 탭 컴포넌트를 사용합니다.
- 활성화 탭은 Royal Blue 아랫선(`border-bottom: 2px solid #2563EB`)과 굵은 텍스트로 강조합니다.

## 4. 재설계 개발 이행 로드맵 (Redesign Implementation)

1. **글로벌 리셋 및 디자인 시스템 수립**:
   - `css/modern-theme.css`, `css/modern-components.css` 생성.
   - 기존의 모든 `xp-` 접두어를 지우고 모던 스타일 클래스인 `modern-` 기반으로 전환.
2. **사이드바 레이아웃 인덱스화**:
   - `index.html`을 데스크톱 화면에서 2열 모던 탐색기 웹 앱으로 전면 교체.
3. **각 도구 페이지 리디자인 적용**:
   - `tools/resize.html`, `tools/crop.html`, `tools/rotate.html`, `tools/jpg-png.html`, `tools/barcode.html`, `tools/gif-maker.html`, `tools/metadata.html`, `tools/video-gif.html` 전체 마크업 및 CSS 리모델링.
   - 비즈니스 로직(JS)은 로직만 분리되어 있으므로 스타일 클래스 선택자 바인딩만 교체하여 개발 리스크 최소화.
   - `barcode.html`에 스캐너 파일 드롭존을 적용하여 바코드/QR코드 이미지 로드 지원.
   - 신규 `pdf.html` (PDF 이미지 상호 변환) 및 `gif-effects.html` (움짤 효과 및 캡션 추가) 개발.
