💡 Cómo usar el Compresor de Imagen

Todo el procesamiento se ejecuta completamente en su navegador usando la API de HTML5 Canvas — ningún archivo se envía a servidores.

  • Control de calidad (10–100): Valores menores reducen más el tamaño pero introducen más imperfecciones en la imagen. Un rango de 70–85 es ideal para la mayoría de las fotos.
  • Formato de salida: Elegir WebP suele ofrecer el mejor equilibrio de compresión y calidad visual en navegadores modernos.
  • Nota sobre PNG: La compresión PNG es sin pérdida, de modo que el deslizador de calidad no afecta a los archivos PNG, solo sus dimensiones y colores determinan el peso.
  • Vista previa en tiempo real: La imagen y las estadísticas se actualizan automáticamente 300 ms después de soltar el control o cambiar el formato.
