💡 How to Use the Image Compressor

All processing happens entirely in your browser using the HTML5 Canvas API — no image data is ever sent to a server.

  • Quality Slider (10–100): Lower values produce smaller files with more visible compression artifacts. 70–85 is the sweet spot for most photos.
  • Output Format: "Same as input" preserves the original format. Switching to WebP typically achieves the best compression-to-quality ratio for modern browsers.
  • PNG note: PNG compression is lossless, so the quality slider has no effect when outputting PNG — only color data and dimensions affect the file size.
  • Live Preview: Preview and size stats update automatically 300 ms after you move the slider or change the format.
  • Savings badge: The "Saved X%" figure is based on the actual compressed blob size returned by the browser — not an estimate.
