🙋 常见问题 (FAQ)

问：在不重新编码的情况下，变速器如何工作？

它解析原始 GIF 二进制文件，定位每帧的图形控制扩展块，并直接修改 2 字节延迟时间值，保留原始图像质量。

问：加快 GIF 速度会改变其文件大小吗？

不会。修改延迟值只会更改元数据，这意味着文件大小与原始文件大小保持完全相同。

问：我可以为各个帧设置不同的速度吗？

此工具将统一的速度倍增器或自定义全局延迟值应用于动画 GIF 的所有帧。
