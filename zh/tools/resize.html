💡 图像调整器

Image Resizer 基于浏览器本机技术构建，利用 Canvas API 2D 直接在客户端应用程序的执行上下文中调整大小。通过集成Hermite/双线性/双三次像素插值，系统执行二元运算并即时处理用户有效负载。该架构消除了传统的往返服务器通信延迟，确保以最佳响应能力和零延迟处理大负载或高频操作。

在现代网页设计中，将光栅图像缩放处理委托给客户端浏览器是速度和安全性方面的重大进步。我们的工具不是在网络通道上来回传递原始数据、图像或媒体流，而是直接在用户的本地线程上运行操作。使用先进的 JavaScript API，该界面可以动态更新视觉状态、更新滑块并立即渲染预览容器，从而创建流畅的、类似桌面的用户体验。

在底层，内存分配是使用 ImageData 像素数组有效处理的。执行周期运行双线性插值数学过滤，以优化数据吞吐量并保持像素或文本保真度。输出的结构严格符合 W3C HTML5 Canvas 标准，确保生成的所有文件、图形或代码段与主流平台、操作系统和专业软件套件完全兼容。

🙋 常见问题 (FAQ)

问：缩放器如何计算新像素？

它将源图像像素网格映射到新的维度，使用插值算法计算新的颜色值并最大限度地减少像素化。

问：双三次插值和双线性插值有什么区别？

双线性插值使用 2x2 像素的加权平均值来提高速度，而双三次插值使用 4x4 像素，以更多处理为代价获得更平滑的边缘。

问：我可以锁定纵横比吗？

是的。调整大小有一个宽高比锁定复选框，以确保图像在调整大小期间不会拉伸或扭曲。
