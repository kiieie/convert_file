💡 이미지 PDF 병합 가이드

모든 이미지 압축 및 PDF 구조 연산은 사용자의 브라우저 로컬 환경에서만 완전히 이루어집니다. 따라서 인터넷 연결 없이도 오프라인에서도 작동합니다.

  • 이미지 로드: "이미지 선택" 단추를 누르거나 드롭존 영역에 변환할 JPG, PNG, WebP, GIF 사진들을 최대 20장까지 올려놓습니다.
  • 페이지 순서 변경: 로드된 사진 썸네일 카드를 마우스 드래그 앤 드롭하여 책장을 넘기듯 좌우상하 순서를 간편하게 재배열할 수 있습니다. 썸네일 왼쪽 위에 페이지 번호가 파란색 뱃지로 노출됩니다.
  • 특정 이미지 제외: 썸네일 오른쪽 위의 빨간 X 버튼을 누르면 해당 사진만 PDF 목록에서 쉽게 지울 수 있습니다.
  • 용지 규격 옵션: A4나 Letter 규격은 종횡비 균형을 맞춰 용지 중앙에 사진을 배치하며, "이미지 원본 크기"는 빈 여백 없이 각 사진 고유의 해상도 규격을 그대로 페이지 크기로 사용합니다.
  • 용지 방향 자동 감지: 사진의 가로/세로 비율을 체크해 한 장 한 장 용지 방향을 다르게 적용해 주는 똑똑한 멀티 레이아웃 기능을 기본 제공합니다.
