여러 개의 PDF 파일들이 업로드된 순서에 맞춰 하나의 큰 PDF 파일로 병합됩니다. 개별 항목의 'X' 버튼을 클릭해 리스트에서 제외할 수 있습니다.

분할 방식 선택
추출할 페이지 범위 콤마(,)와 하이픈(-)으로 입력 (예: 2, 4-6)
불필요 메타데이터 및 객체 최적화 압축 저장
페이지 삭제 삭제하여 제외할 페이지 범위 입력
페이지 회전 회전을 적용할 페이지 범위와 각도 선택
텍스트 워터마크 추가
폰트 크기
투명도 (0.1 ~ 1.0)
모든 페이지 중앙에 지정한 영문/숫자 텍스트를 사선으로 배치
모든 페이지 하단 중앙에 "페이지 번호 (현재페이지 / 전체페이지)" 삽입

💡 PDF 변환기 가이드

인터넷 중계 및 전송 없이 브라우저 메모리 자원으로만 100% 무손실 처리되는 클라이언트 사이드 전용 보안 변환 툴입니다.

  • 이미지 병합: 업로드 순서대로 PDF 페이지가 생성되며, 필요시 우상단 'X' 버튼을 눌러 목록에서 지울 수 있습니다.
  • PDF 추출: 다량의 페이지를 가진 PDF 문서는 각 페이지를 고화질 이미지화하여 단일 ZIP 압축파일로 번들링 제공합니다.
  • PDF 끼리 병합: 업로드한 순서에 맞춰 두 개 이상의 PDF 문서들을 순차 결합하여 하나의 PDF 파일로 합칩니다.
  • PDF 페이지 분할: 전체 페이지를 1장짜리 개별 파일들로 한 번에 분할(ZIP)하거나, `2,4-6` 형태로 원하는 범위만 골라 새 PDF로 추출합니다.
  • PDF 편집: 불필요한 메타데이터 제거 압축, 지정 페이지 완전 삭제 및 시계방향 회전, 워터마크 추가, 페이지 번호 매기기를 결합 수행합니다.
